Gemini AI API Documentation

Welcome to the Gemini AI API documentation. This API allows you to interact with our AI model for chat and code conversion capabilities.

Authentication

API Key

All requests require an API key passed in the headers:

{
  "Authorization": "Bearer YOUR_API_KEY"
}

Chat Endpoint

POST /api/chat

Send a message to the AI and receive a response.

Request

{
  "message": "Hello, how can you help me?",
  "context": "optional previous messages for context"
}

Response

{
  "response": "I'm an AI assistant powered by Gemini...",
  "timestamp": 1645564789000
}

Example Usage

fetch('https://your-api.com/api/chat', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_API_KEY'
  },
  body: JSON.stringify({
    message: 'Hello, how can you help me?'
  })
})

Code Conversion

POST /api/convert

Convert code between different programming languages.

Request

{
  "code": "your code here",
  "fromLanguage": "python",
  "toLanguage": "javascript"
}

Response

{
  "convertedCode": "converted code here",
  "language": "javascript"
}

Examples

Chat Example

import { GeminiService } from './services/gemini.service';

const gemini = new GeminiService('YOUR_API_KEY');

async function chat() {
  try {
    const response = await gemini.generateResponse(
      'Explain how to use the Gemini API'
    );
    console.log(response);
  } catch (error) {
    console.error('Error:', error);
  }
}
